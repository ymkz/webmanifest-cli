# webmanifest-cli

> A generator of webmanifest and icons

[![](https://github.com/ymkz/webmanifest-cli/workflows/check-pull-request/badge.svg)](https://github.com/ymkz/webmanifest-cli/actions?query=workflow%3A"check-pull-request")
[![](https://github.com/ymkz/webmanifest-cli/workflows/healthcheck-master/badge.svg)](https://github.com/ymkz/webmanifest-cli/actions?query=workflow%3A"healthcheck-master")

## Prerequisites

- node >=12

## Install

```sh
npm i -D webmanifest-cli
```

## Usage

```sh
$ webmanifest
$ webmanifest --icon public/icon.svg --manifest public/manifest.webmanifest --outputIcon public/assets --outputManifest public/manifest.webmanifest --sizes 512,384,192,180,152,144,128,96,72
```
